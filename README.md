# masakoha.github.io

静的な個人 Web ページです。`index.html` を開くだけで表示できます。

## Files

- `index.html`: ページ本体
- `styles.css`: スタイル定義
- `script.js`: スクロール時の表示アニメーション

## Publish

GitHub Pages の User Pages として公開します。

- `https://masakoha.github.io/`
- `https://masakoha.github.io/index.html`

イベントページも同じリポジトリ配下で公開します。
