# www.masakoha.com

静的な個人 Web ページです。`index.html` を開くだけで表示できます。

## Files

- `index.html`: ページ本体
- `styles.css`: スタイル定義
- `script.js`: スクロール時の表示アニメーション

## Publish

GitHub Pages でカスタムドメイン公開します。

- `https://www.masakoha.com/`
- `https://www.masakoha.com/index.html`

イベントページも同じリポジトリ配下で公開します。
